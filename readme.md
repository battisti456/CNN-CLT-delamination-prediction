# Ultrasonic Prediction of Delamination in Cross-laminated Timber using CNNs

This repository includes the written report and the code used for AI535: Deep Learning at Oregon State University.

The focus of the project was to use deep learning in a novel way. For this project, the focus of predicting delamination test results in cross-laminated timber (CLT) using through-transmission ultrasound was selected. The report, [`report.pdf`](report.pdf) details the various attempts at modeling the problem.

This project found that deep learning did not perform any better than non-deep-learning techniques and that no techniques performed significantly well for this data-set.
